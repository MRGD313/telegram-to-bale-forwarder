# Contributing

1. Fork the repo and create a branch.
2. Do not commit `.env`, `session*`, or `*.db` files.
3. Test with `scripts/run_e2e_from_zero.ps1` if you change send/crawl logic.
4. Open a PR with a short description of behavior change.

Issues welcome: forum edge cases, Bale API quirks, and compression defaults for different networks.
